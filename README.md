# Data-Scrapper
Data Scrapper for Foclonline
DataScrapper v1.0

# How to use:
1. Open the file.
2. Copy everything from the file.
3. Open the console on the webpage where you want to pull the data from.
4. Hit CTRL + V or Paste in the Console then Hit Enter
5. Enjoy
